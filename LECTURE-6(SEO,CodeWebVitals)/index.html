CODE WEB VITALS

Code web vitals are an initiative by Google to provide unified guidance for quality signals that are essential to delivering a great user experience on the web.

CLS,LCP,FID(all are metric reports):

CLS-cummulative layout shift: it is the measure of largest burst of layout shift scores for every unexpected layout shift.

LCP-largest contextual point: it is a metric report that render the time of the largest image(loading time of a largest image/element).In at least 2.5 seconds the largest element of the page should be loaded.

FID-first input delay: it is the time from where the user first interacts with a page.To provide a good experience to the user the site should load at 100 milliseconds.

HOW TO GENERATE A LIGHT HOUSE REPORT:

  1. Right click on the screen.
  2. Click on inspect(from drop down menu)
  3. click on light house.(view the light house report)
